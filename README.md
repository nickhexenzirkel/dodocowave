# 🎮 DODOCO WAVE

Jogo de ritmo estilo Guitar Hero com músicas do YouTube.

---

## ⚙️ Instalação

### Pré-requisito
- **Node.js** (v18+) → https://nodejs.org/

### Passos

```
1. Extraia o .zip
2. Abra o CMD/terminal dentro da pasta
3. npm install
4. npm start
```

> Na primeira música, o app baixa o `yt-dlp.exe` automaticamente (~10MB).
> Da segunda vez em diante, abre instantaneamente.

---

## 🎮 Controles

| Tecla | Ação |
|-------|------|
| A | Lane 1 (Ciano) |
| S | Lane 2 (Magenta) |
| D | Lane 3 (Roxo) |
| F | Lane 4 (Amarelo) |
| P | OVERDRIVE (encha a barra primeiro!) |
| ESC | Terminar fase |

### 🎸 Notas Hold (Guitar Hero)
- **Círculo** = toque rápido
- **Diamante** = segure até o arco completar!

---

## 🐛 Problemas comuns

| Erro | Solução |
|------|---------|
| `Cannot find module` | Rode `npm install` |
| `Falha ao baixar yt-dlp` | Verifique conexão com internet |
| Vídeo não carrega | Tente outro vídeo (alguns têm DRM) |
